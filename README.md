# TP React — Composants et Props

## Description
Application React réalisée dans le cadre d'un TP pédagogique.
Elle couvre les bases de React : composants fonctionnels, props et PropTypes.

## Composants créés

| Composant | Description |
|---|---|
| `HelloWorld` | Affiche un message de bienvenue |
| `Greeting` | Salue un utilisateur par son prénom |
| `Profil` | Affiche la photo et le nom d'un utilisateur |
| `Voiture` | Affiche les informations d'une voiture |
| `ListeCourses` | Affiche une liste dynamique d'éléments |

## Concepts abordés
- Composants fonctionnels JSX
- Passage de données via les **props**
- Validation des props avec **PropTypes**
- Valeurs par défaut avec **defaultProps**
- Rendu de listes avec `.map()`

## Installation et lancement
```bash
# Cloner le projet
git clone https://github.com/votre-username/tp-react.git

# Entrer dans le dossier
cd tp-react

# Installer les dépendances
npm install

# Lancer l'application
npm start
```

L'application s'ouvre sur **http://localhost:3000**
